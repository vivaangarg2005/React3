# Profile Dashboard Cards 🪪

A sleek, responsive team member dashboard built with **React** and **Lucide React**.  
This project replicates a modern UI design featuring profile cards with dynamic statistics and an automated member rating system.

---

## 🚀 Features

### ✅ Dynamic Rating Bar  
The progress bar automatically calculates its width based on the `points` prop relative to a maximum score.

### ✅ Responsive Grid  
Cards are organized using CSS Flexbox/Grid for a clean layout across all screen sizes.

### ✅ Modern Styling  
Subtle shadows, clean typography, and a soft color palette inspired by professional SaaS dashboards.

### ✅ Reusable Components  
A single `Card` component that renders user data dynamically via props.

---

## 🛠️ Tech Stack

- **React** – UI Library  
- **Lucide React** – Clean, consistent iconography (`Mail`, `Star`)  
- **CSS3** – Custom styling with Flexbox, spacing, and layout control
